# 项目文件

| 文件名          | 功能                               |
| --------------- | ---------------------------------- |
| transformer.py | 定义 Transformer 模型的主体结构。      |
| encoding.py     | 包含位置编码 (Positional Encoding) 的实现。 |
| train.py        | 包含模型训练的代码。                    |
| inference.py    | 包含模型推理的代码。                    |
| test.py         | 包含模型测试的代码。                    |
